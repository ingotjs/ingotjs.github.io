Why Ingot?

Every package follows the same principles — type-safe, AI-navigable, production-ready.

AI-First Architecture

42 Claude Code skills, comprehensive CLAUDE.md instructions, and strict code standards that eliminate ambiguity. AI writes better code here than in most hand-rolled projects.

Production-Ready

Auth, API, database, email, i18n, observability, and CI/CD — all wired up and working together. Not a skeleton — a foundation you can ship on.

Edge-Native

Cloudflare Workers + D1 + KV. Your app runs on the same global edge network as your database. Sub-10ms reads, zero egress fees, global replication.

i18n From Day One

Every user-facing string flows through Lingui — UI, auth errors, emails, and meta tags. Adding a language is a PO file, not a refactor.

Bulletproof Quality

Oxlint + Oxfmt + tsgo + syncpack + Knip + Playwright E2E. Pre-commit hooks, supply chain scanning, and one command (bun ok) that validates everything.

Deploy in 60 Seconds

Alchemy IaC provisions D1, applies migrations, and deploys the Worker — all from one TypeScript file. CI auto-deploys on push to main.
